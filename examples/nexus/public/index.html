About Nexus

Nexus is a semantic server framework built on WFL's native server capabilities. It follows a simple Server -> Router architecture designed for clarity and ease of use.

Features

  • Static File Serving - Automatically serves files from the /public directory
  • Dynamic Routing - Register custom handlers for API endpoints
  • Pretty URLs - Supports .html extension-free URLs
  • Security - Built-in path sanitization to prevent directory traversal

Try the API

Test the dynamic routes:

  • /api/hello - Simple API endpoint
  • /api/users - List of users
  • /health - Health check endpoint
  • /api/echo - Echo request info
