## 2024-05-18 - [Optimize percent_decode using Cow for zero-cost fast paths]
**Learning:** In URL decoding (and string processing in general), it's highly beneficial to avoid string allocation entirely if the string does not need mutation. Rust's `std::borrow::Cow` allows us to return a borrowed string slice when no decoding is necessary and an owned `String` when mutation happens, eliminating unnecessary heap allocations. Using a fast-path scan `!bytes.iter().any(|&b| b == b'%' || b == b'+')` is very efficient.
**Action:** When implementing text manipulation or decoding functions that may return the input unaltered in common cases, use `Cow<'a, str>` (or `Arc<str>` fast paths) combined with a fast byte scan to skip allocations entirely.
