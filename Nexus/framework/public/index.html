NEXUS

A lightweight, semantic server framework for WebFirst Language

API Endpoints

GET / This page (static file)
GET /hello Simple greeting
GET /api/status Health check
GET /api/time Server time
GET /api/info Request info
POST /api/echo Echo body back

Architecture

// Request Flow
Client Request -> Server (Bouncer)
              -> Router (Traffic Cop)
                  -> Check 1: Static Files (Lazy)
                  -> Check 2: Dynamic Routes
                  -> Check 3: 404 Fallback
              -> Handler (Talent)
              <- Response
